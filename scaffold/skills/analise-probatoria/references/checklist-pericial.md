# Checklist: Prova Pericial

> Referência para avaliação de laudos periciais e pareceres técnicos.
> Base legal: CPC arts. 464-480, CPP arts. 158-184.

## Tipos de Perícia

| Tipo | Exemplos |
|------|----------|
| Médica | Lesão corporal, capacidade laboral, insanidade, causa mortis |
| Contábil | Lucros, balanços, cálculos financeiros |
| Grafotécnica | Autenticidade de assinaturas, falsificação |
| Engenharia | Avaliação de imóveis, causas de acidentes |
| Informática forense | Análise de dispositivos, recuperação de dados |
| Genética (DNA) | Identificação, parentesco |
| Toxicológica | Substâncias no organismo |
| Balística | Projéteis, armas, trajetória |

## Checklist de Avaliação

### A. Validade Formal
- [ ] Perito regularmente nomeado pelo juízo?
- [ ] Habilitação profissional na área (registro em conselho)?
- [ ] Assistentes técnicos oportunizados às partes?
- [ ] Quesitos formulados e respondidos?
- [ ] Laudo apresentado no prazo legal?
- [ ] Contraditório sobre o laudo garantido?

### B. Qualidade Metodológica (Critérios Daubert Adaptados)
- [ ] Método cientificamente testável e falsificável?
- [ ] Publicado e revisado por pares (peer review)?
- [ ] Taxa de erro conhecida e aceitável?
- [ ] Protocolos e padrões operacionais reconhecidos seguidos?
- [ ] Aceito pela comunidade científica da área?

### C. Cadeia de Custódia (CPP arts. 158-A a 158-F)
- [ ] Documentada desde a coleta?
- [ ] Registro de todos os responsáveis pelo manuseio?
- [ ] Condições de acondicionamento e transporte adequadas?
- [ ] Lacres íntegros documentados?
- [ ] Quebra identificada? Se sim, qual o impacto?

### D. Conteúdo do Laudo
- [ ] Descreve claramente o objeto da perícia?
- [ ] Indica a metodologia utilizada?
- [ ] Conclusões fundamentadas nos dados observados?
- [ ] Responde a todos os quesitos?
- [ ] Conclusões claras, sem ambiguidades?

### E. Divergências
- [ ] Há parecer de assistente técnico divergente?
- [ ] Fundamentação de cada posição identificada?
- [ ] Pontos de divergência específicos mapeados?

### F. Exame de Corpo de Delito (Processo Penal)
- [ ] Infração deixa vestígios?
- [ ] Se sim: exame de corpo de delito realizado (CPP art. 158)?
- [ ] **REGRA:** Confissão NÃO supre falta do exame de corpo de delito

### G. Classificação

| Classificação | Critério |
|---------------|----------|
| **ALTA** | Daubert 5/5 + cadeia íntegra + sem divergência |
| **MÉDIA** | Daubert 3-4/5 OU divergência parcial OU cadeia parcial |
| **BAIXA** | Daubert <3/5 OU quebra de cadeia OU divergência grave |

## Referências
- Daubert v. Merrell Dow Pharmaceuticals, 509 U.S. 579 (1993)
- Lei 13.964/2019 (cadeia de custódia)
