## 1. REGRA FUNDAMENTAL: EC 113/2021

A partir de **dezembro/2021**, todas as dívidas da Fazenda Pública passam a ser atualizadas **exclusivamente pela SELIC**.

**IMPORTANTE**: A SELIC já engloba correção monetária + juros. **NÃO acumular** com outros índices.

## 9. HONORÁRIOS ADVOCATÍCIOS

### 9.1 Base de Cálculo

| Situação | Regra |
|----------|-------|
| Fixados sobre valor da causa | Atualizar desde ajuizamento (Súmula 14/STJ) |
| Fixados sobre valor da condenação | Aplicar percentual sobre valor atualizado |
| Fixados em valor certo | Atualizar desde decisão que arbitrou |

### 9.2 Atualização

- **Correção**: IPCA-E (até nov/2021), SELIC (a partir de dez/2021)
- **Juros**: A partir do trânsito em julgado

### 9.3 Honorários em Cumprimento de Sentença

- **Taxa**: 10% (não reduzível)
- **Sobre**: Débito atualizado (excluída a multa de 10% do art. 523)

## 10. CONSOLIDAÇÃO EM DEZ/2021 (Regra de Transição)

### Procedimento para prestações até dez/2021:

1. Consolidar crédito em dez/2021 pelos critérios até então aplicáveis
2. Considerar INPC de nov/2021 (0,84%) e juros de dez/2021 (0,4412%)
3. Sobre o valor consolidado, aplicar SELIC a partir de jan/2022
4. Somar "Juros até 12/2021" + "Juros Selic" para total

### Forma de apresentação dos cálculos:

| Coluna | Descrição |
|--------|-----------|
| Principal corrigido | Valor atualizado até dez/2021 |
| Juros até 12/2021 | Juros pelos critérios anteriores |
| Juros Selic | SELIC aplicada a partir de jan/2022 |
| **Total** | Principal + Juros até 12/2021 + Juros Selic |

## 12. ERROS COMUNS A EVITAR

| Erro | Correção |
|------|----------|
| Usar TR em benefício previdenciário | Substituir por INPC (até nov/21) ou SELIC (dez/21+) |
| Acumular SELIC + outro índice | Usar APENAS SELIC (já inclui correção + juros) |
| Acumular SELIC + 1% ao mês | Usar APENAS SELIC |
| Não especificar marco temporal | Definir: "desde a citação", "desde o DIB", etc. |
| Usar "conforme legislação" | Especificar índice e marco concretos |
| Juros compostos (capitalização) | SELIC é capitalização simples |
| Esquecer consolidação em dez/2021 | Aplicar regra de transição |
